Curso Javascript: Uso de prompt y Ejercicio de ejemplo.

Ejercicio donde se pone a prueba lo visto hasta el momento en el curso de Javascript.

Detalles del ejercicio:

  1. El usuario debera adivinar un numero que se generara de forma aleatoria del 1 al 100.
  2. Al cargar la pagina HTML, mostrara una ventana emergente en pantalla.
  3. El usuario debera escribir un numero entre el 1 y el 100.
  4. Si el usuario adivina, mostrara un mensaje de enhorabuena y finalazara la aplicacion.
  5. Si el usuario no adivina, se le invitara a escribir nuevamente un numero.
  6. Si el usuario escribe 0, finaliza la aplicacion mostrando un mensaje de despedida.
  7. Cualquier otro caracter diferente a un numero o que sea mayor a 100 o menor a 0, mostrara un error y se debera volver a intentar.
